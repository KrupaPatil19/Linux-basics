# Basic Linux Commands

## pwd
Displays the current working directory.

Example:
```bash
pwd
```

---

## ls
Lists files and directories.

Example:
```bash
ls
ls -l
ls -a
```

---

## cd
Changes the current directory.

Example:
```bash
cd Documents
cd ..
cd ~
```

---

## mkdir
Creates a new directory.

Example:
```bash
mkdir project
```

---

## rmdir
Removes an empty directory.

Example:
```bash
rmdir project
```
